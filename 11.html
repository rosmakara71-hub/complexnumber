рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бҹўбҹҘбҹЎ. бһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \frac{(1+i)^2}{1-i}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶ $(1+i)^2 = 2i$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $z = \frac{2i}{1-i}$бҹ”
  • бһӮбһ»бһҺбһ“бһ№бһ„бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ– $z = \frac{2i(1+i)}{(1-i)(1+i)} = \frac{2i + 2i^2}{1+1} = \frac{-2 + 2i}{2} = -1 + i$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһӮбһә $-1$бҹ”
бҹўбҹҘбҹў. бһ”бһҫ $z = \frac{1+i\sqrt{3}}{2}$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $z \cdot \overline{z}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z \cdot \overline{z} = |z|^2$бҹ”
  • $|z| = \sqrt{\left(\frac{1}{2}\right)^2 + \left(\frac{\sqrt{3}}{2}\right)^2} = \sqrt{\frac{1}{4} + \frac{3}{4}} = 1$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $|z|^2 = 1^2 = 1$бҹ”
бҹўбҹҘбҹЈ. бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $i^{-103}$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбһ¶бһҳбһүбҹ’бһүбһ”бҹҶбһ•бһ»бһҸбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^{-103} = \frac{1}{i^{103}}$бҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡ $103 = 4 \times 25 + 3$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $i^{103} = i^3 = -i$бҹ”
  • бһҖбһ“бҹ’бһҹбҹ„бһҳбһ‘бҹ…бһҮбһ¶ $\frac{1}{-i} = \frac{i}{-i^2} = \frac{i}{1} = i$бҹ”
бҹўбҹҘбҹӨ. бһҡбһҖбһҳбһјбһҢбһ»бһӣбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \frac{(3+4i)^2}{1+i}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = \frac{|3+4i|^2}{|1+i|}$ бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһјбһҢбһ»бһӣбҹ”
  • $|3+4i| = \sqrt{3^2+4^2} = 5$ бһ“бһ·бһ„ $|1+i| = \sqrt{1^2+1^2} = \sqrt{2}$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $|z| = \frac{5^2}{\sqrt{2}} = \frac{25}{\sqrt{2}} = \frac{25\sqrt{2}}{2}$бҹ”
бҹўбҹҘбҹҘ. бһ”бһҫ $x, y \in \mathbb{R}$ бһ бһҫбһҷ $(x+yi)(2-3i) = 4+i$ бһҸбһҫбһҸбһҳбҹ’бһӣбҹғ $x$ бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһ¶бһүбһҡбһҖ $x+yi = \frac{4+i}{2-3i} = \frac{(4+i)(2+3i)}{(2-3i)(2+3i)} = \frac{8+12i+2i-3}{4+9} = \frac{5+14i}{13}$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $x = \frac{5}{13}$ (бһ“бһ·бһ„ $y = \frac{14}{13}$)бҹ”
бҹўбҹҘбҹҰ. бһҹбһҡбһҹбҹҒбһҡ $z = \sqrt{3} - i$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ…бһҳбҹ’бһ”бһ„бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $r = \sqrt{3+1} = 2$бҹ” бһ…бҹҶбһҺбһ»бһ…бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹӨ ($x>0, y<0$) бһ“бһ¶бҹҶбһұбҹ’бһҷ $\theta = \arctan\left(\frac{-1}{\sqrt{3}}\right) = -\frac{\pi}{6}$ бһҠбҹӮбһӣбһҮбһ¶бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһҖбҹ’бһ“бһ»бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮ $(-\pi, \pi]$бҹ”
бҹўбҹҘбҹ§. бһ”бһҫ $z = \cos\frac{2\pi}{7} + i\sin\frac{2\pi}{7}$ бһӮбһҺбһ“бһ¶бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһ“бҹғбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһ…бҹ’бһҡбһ¶бһҹ $\arg(z^{-1})$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбҹ– $\arg(z^{-1}) = -\arg(z)$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $\arg(z^{-1}) = -\frac{2\pi}{7}$бҹ”
бҹўбҹҘбҹЁ. бһҖбҹҶбһҺбһҸбҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ“бҹғ $z = 5i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = 5i$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһјбһҢбһ»бһӣ $r = 5$ бһ“бһ·бһ„бһҳбһ»бҹҶбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢ $\theta = \frac{\pi}{2}$бҹ”
бҹўбҹҘбҹ©. бһҹбһҳбҹ’бһҡбһҪбһӣбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡ $z = -2\left(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4}\right)$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң (бһҳбһјбһҢбһ»бһӣбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ”бһ¶бһҸбҹӢбһҹбһүбҹ’бһүбһ¶бһҠбһҖбһ“бҹ…бһ–бһёбһҳбһ»бһҒбһҳбһјбһҢбһ»бһӣ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бһјбһҖбһҳбһ»бҹҶбһ“бһ№бһ„ $\pi$ (бһ¬бһҠбһҖ $\pi$ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһҳбһ»бҹҶбһ…бһҳбҹ’бһ”бһ„)бҹ”
  • $\theta = \frac{\pi}{4} - \pi = -\frac{3\pi}{4}$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $2\left(\cos\left(-\frac{3\pi}{4}\right) + i\sin\left(-\frac{3\pi}{4}\right)\right)$бҹ”
бҹўбҹҰбҹ . бһ”бһҫ $z_1 = r_1(\cos\theta_1 + i\sin\theta_1)$ бһ“бһ·бһ„ $z_2 = r_2(\cos\theta_2 + i\sin\theta_2)$ бһҸбһҫбһҖбһ“бҹ’бһҹбҹ„бһҳ $\arg\left(\frac{\overline{z_1}}{z_2}\right)$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\arg\left(\frac{\overline{z_1}}{z_2}\right) = \arg(\overline{z_1}) - \arg(z_2)$бҹ”
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $\arg(\overline{z_1}) = -\theta_1$ бһ“бһ·бһ„ $\arg(z_2) = \theta_2$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһә $-\theta_1 - \theta_2$бҹ”
бҹўбҹҰбҹЎ. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\left(\frac{1+i\sqrt{3}}{1-i\sqrt{3}}\right)^{10}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһ„ $w = \frac{1+i\sqrt{3}}{1-i\sqrt{3}} = \frac{2(\cos\frac{\pi}{3}+i\sin\frac{\pi}{3})}{2(\cos(-\frac{\pi}{3})+i\sin(-\frac{\pi}{3}))} = \cos\frac{2\pi}{3} + i\sin\frac{2\pi}{3}$бҹ”
  • бһҸбһ¶бһҳбһҠбһәбһҳбҹҗбһҡбҹ– $w^{10} = \cos\frac{20\pi}{3} + i\sin\frac{20\pi}{3}$бҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡ $\frac{20\pi}{3} = 6\pi + \frac{2\pi}{3} \equiv \frac{2\pi}{3}$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $w^{10} = \cos\frac{2\pi}{3} + i\sin\frac{2\pi}{3} = -\frac{1}{2} + i\frac{\sqrt{3}}{2}$бҹ”
бҹўбҹҰбҹў. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1-i)^{16}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-i)^2 = -2i$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $(1-i)^{16} = ((-2i)^2)^4 = (-4)^4 = 256$бҹ”
бҹўбҹҰбҹЈ. бһ”бһҫ $z - \frac{1}{z} = 2i\sin\theta$ бһҸбһҫбһҖбһ“бҹ’бһҹбҹ„бһҳ $z^n - \frac{1}{z^n}$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҠбһҫбһҳбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹ„бһҷ $z = \cos\theta + i\sin\theta$бҹ”
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһҠбһәбһҳбҹҗбһҡ $z^n = \cos n\theta + i\sin n\theta$ бһ“бһ·бһ„ $z^{-n} = \cos n\theta - i\sin n\theta$бҹ”
  • бһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҳбһ¶бһҺбһңбһ·бһ’бһёбһҠбһҖбҹ– $z^n - z^{-n} = 2i\sin n\theta$бҹ”
бҹўбҹҰбҹӨ. бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ $n$ бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷ $\left(\frac{1+i}{1-i}\right)^n = 1$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бҹ– $\frac{1+i}{1-i} = \frac{(1+i)^2}{2} = \frac{2i}{2} = i$бҹ”
  • бһҹбһҳбһёбһҖбһ¶бһҡбһ‘бҹ…бһҮбһ¶ $i^n = 1$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҸбһҳбҹ’бһӣбҹғбһӮбһҸбҹӢбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһ“бҹғ $n$ бһӮбһә бҹӨбҹ”
бҹўбҹҰбҹҘ. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(\sqrt{3}+i)^6$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһӮбһә $2\left(\cos\frac{\pi}{6} + i\sin\frac{\pi}{6}\right)$бҹ”
  • бһҸбһ¶бһҳбһҠбһәбһҳбҹҗбһҡбҹ– $2^6\left(\cos\left(6 \times \frac{\pi}{6}\right) + i\sin\left(6 \times \frac{\pi}{6}\right)\right) = 64(\cos\pi + i\sin\pi) = 64(-1+0) = -64$бҹ”
бҹўбҹҰбҹҰ. бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -7 + 24i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһ„бһ«бһҹбһҖбһ¶бһҡбҹҒ $x+yi$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $x^2-y^2 = -7$ бһ“бһ·бһ„ $2xy = 24 \Rightarrow xy = 12 > 0$ ($x,y$ бһҹбһүбҹ’бһүбһ¶бһҠбһјбһ…бһӮбҹ’бһ“бһ¶)бҹ”
  • бһҳбһјбһҢбһ»бһӣ $x^2+y^2 = \sqrt{(-7)^2+24^2} = 25$бҹ”
  • бһ”бһјбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ”бһ¶бһ“ $2x^2 = 18 \Rightarrow x = \pm 3$бҹ” бһҮбҹҶбһ“бһҪбһҹбһҡбһҖ $y = \pm 4$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm(3+4i)$бҹ”
бҹўбҹҰбҹ§. бһ”бһҫ $\omega$ бһҮбһ¶бһ«бһҹбһӮбһјбһ”бһҳбһ·бһ“бһ–бһ·бһҸбһ“бҹғ $1$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1-\omega+\omega^2)(1+\omega-\omega^2)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲ $1+\omega+\omega^2 = 0$бҹ– $1+\omega^2 = -\omega$ бһ“бһ·бһ„ $1+\omega = -\omega^2$бҹ”
  • бһҮбҹҶбһ“бһҪбһҹбһ…бһјбһӣбҹ– $(-\omega - \omega)(-\omega^2 - \omega^2) = (-2\omega)(-2\omega^2) = 4\omega^3 = 4(1) = 4$бҹ”
бҹўбҹҰбҹЁ. бһҸбһҫбһ•бһӣбһ”бһјбһҖбһ“бҹғбһ«бһҹбһ‘бһё $n$ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z$ ($z \neq 0, n \geq 2$) бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ…бҹҶбһҺбһ»бһ…бһҸбҹҶбһҺбһ¶бһ„бһ«бһҹбһ‘бһё $n$ бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бһҮбһ¶бһ–бһ бһ»бһҖбҹ„бһҺбһ“бһ·бһ‘бҹҗбһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ…бһ·бһҸбһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӮбһӣбҹӢбһҸбһҳбҹ’бһҡбһ»бһҷ $(0,0)$ бһҮбһ¶бһ“бһ·бһ…бҹ’бһ… бһҠбһјбһ…бһ“бҹҒбҹҮбһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһ’бһҡбһҺбһёбһҳбһ¶бһҸбҹ’бһҡ бһ¬бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $w^n - z = 0$ (бһҳбҹҒбһӮбһ»бһҺбһҸбһҪ $w^{n-1}$ бһҹбҹ’бһҳбһҫ бҹ ) бһ•бһӣбһ”бһјбһҖбһ«бһҹбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбһәбһҹбҹ’бһҳбһҫ $0$бҹ”
бҹўбҹҰбҹ©. бһҡбһҖбһ«бһҹбһ‘бһё бҹў бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -i$ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = \cos\left(-\frac{\pi}{2}\right) + i\sin\left(-\frac{\pi}{2}\right)$бҹ”
  • бһ«бһҹбһҖбһ¶бһҡбҹҒбһ‘бһёбҹЎбҹ– $\cos\left(-\frac{\pi}{4}\right) + i\sin\left(-\frac{\pi}{4}\right) = \frac{\sqrt{2}}{2} - i\frac{\sqrt{2}}{2} = \frac{1-i}{\sqrt{2}}$бҹ”
  • бһ«бһҹбһҖбһ¶бһҡбҹҒбһ‘бһёбҹў бһӮбһәбһ•бҹ’бһ‘бһ»бһҷбһ“бһ№бһ„бһ«бһҹбһ‘бһёбҹЎ бһҠбһјбһ…бһ“бҹҒбҹҮбһ…бһҳбҹ’бһӣбһҫбһҷбһӮбһә $\pm \frac{1-i}{\sqrt{2}}$бҹ”
бҹўбҹ§бҹ . бһҡбһҖбһ•бһӣбһӮбһ»бһҺбһ“бҹғбһ«бһҹбһ‘бһё бҹҘ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғ $1$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһӮбһ»бһҺбһ«бһҹбһ‘бһё $n$ бһ“бҹғ $1$ бһӮбһәбһҹбҹ’бһҳбһҫ $(-1)^{n-1}$бҹ”
  • бһ…бҹҶбһ–бҹ„бҹҮ $n=5$бҹ– бһ•бһӣбһӮбһ»бһҺбһӮбһә $(-1)^{5-1} = (-1)^4 = 1$бҹ”
бҹўбҹ§бҹЎ. бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + 4 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^2 = -4 \Rightarrow z = \pm \sqrt{-4} = \pm 2i$бҹ”
бҹўбҹ§бҹў. бһ”бһҫ $z = 1-2i$ бһҮбһ¶бһ«бһҹбһҳбһҪбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + bz + c = 0$ (бһҠбҹӮбһӣ $b, c \in \mathbb{R}$) бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $b$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҠбҹ„бһҷбһҹбһ¶бһҡбһҳбҹҒбһӮбһ»бһҺбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ бһ«бһҹбһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбһә $1+2i$бҹ”
  • бһҸбһ¶бһҳбһңбҹҖбһҸ бһ•бһӣбһ”бһјбһҖбһ«бһҹ $S = (1-2i) + (1+2i) = 2$бҹ”
  • бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҳбҹҒбһӮбһ»бһҺбҹ– $S = -b \Rightarrow 2 = -b \Rightarrow b = -2$бҹ”
бҹўбҹ§бҹЈ. бһҡбһҖбһҹбҹҶбһҺбһ»бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^3 + z = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҠбһ¶бһҖбҹӢ $z$ бһҮбһ¶бһҖбһҸбҹ’бһҸбһ¶бһҡбһҪбһҳбҹ– $z(z^2 + 1) = 0$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $z = 0$ бһ¬ $z^2 = -1 \Rightarrow z = \pm i$бҹ” бһҹбҹҶбһҺбһ»бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷбһӮбһә $\{0, i, -i\}$бҹ”
бҹўбҹ§бҹӨ. бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… бһҹбһҳбһёбһҖбһ¶бһҡ $|z - (1+2i)| = 3$ бһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһҒбҹ’бһҹбҹӮбһҖбҹ„бһ„бһ’бһҡбһҺбһёбһҳбһ¶бһҸбҹ’бһҡбһҳбһҪбһҷбһҺбһ¶? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бһјбһ‘бҹ… $|z - z_0| = R$ бһӮбһәбһҮбһ¶бһҡбһ„бҹ’бһңбһ„бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ…бһ·бһҸ $z_0$ бһ“бһ·бһ„бһҖбһ¶бҹҶ $R$бҹ”
  • бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ $z_0 = 1+2i$ бһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҖбһјбһўбһҡбһҠбҹ„бһ“бҹҒбһ•бҹ’бһ…бһ·бһҸ $(1,2)$ бһ“бһ·бһ„бһҖбһ¶бҹҶ $R=3$бҹ”
бҹўбҹ§бҹҘ. бһ”бһҫ $\alpha$ бһ“бһ·бһ„ $\beta$ бһҮбһ¶бһ«бһҹбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + z + 1 = 0$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\alpha^3 + \beta^3$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2+z+1=0$ бһӮбһәбһҮбһ¶бһ«бһҹбһӮбһјбһ”бһҳбһ·бһ“бһ–бһ·бһҸбһ“бҹғ $1$ (бһ–бҹ’бһҡбҹ„бҹҮ $(z-1)(z^2+z+1) = z^3-1 = 0$)бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $\alpha^3 = 1$ бһ“бһ·бһ„ $\beta^3 = 1$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\alpha^3 + \beta^3 = 1 + 1 = 2$бҹ”
