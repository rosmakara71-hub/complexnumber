рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $(1+i)^4 - (1-i)^4$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҷбһҫбһ„бһҳбһ¶бһ“ $(1+i)^2 = 2i$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $(1+i)^4 = (2i)^2 = -4$бҹ”
  • бһ бһҫбһҷ $(1-i)^2 = -2i$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $(1-i)^4 = (-2i)^2 = -4$ Lynnбҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ бһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһә $-4 - (-4) = 0$бҹ”
бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ $x$ бһ“бһ·бһ„ $y$ бһҠбҹӮбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһёбһҖбһ¶бһҡ $(x+y) + (x-y)i = 5 + 3i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ•бҹ’бһ‘бһ№бһҳбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһ“бһ·бһ„бһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбҹ– $x+y=5$ бһ“бһ·бһ„ $x-y=3$бҹ”
  • бһ”бһјбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ”бһ¶бһ“ $2x = 8 \Rightarrow x=4$бҹ”
  • бһ‘бһ¶бһүбһҡбһҖ $y = 5 - 4 = 1$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{1}{1+i} + \frac{1}{1-i}$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһҳбҹ’бһҡбһјбһңбһ—бһ¶бһӮбһ”бҹӮбһ„бһҡбһҪбһҳбҹ– $\frac{(1-i) + (1+i)}{(1+i)(1-i)} = \frac{2}{1 - i^2} = \frac{2}{2} = 1$бҹ”
бһ”бһҫ $z = 3 - 4i$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $z^2 - 6z$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ–бһё $z = 3 - 4i \Rightarrow z - 3 = -4i$бҹ”
  • бһӣбһҫбһҖбһҮбһ¶бһҖбһ¶бһҡбҹҒбһ‘бһ¶бҹҶбһ„бһҹбһ„бһҒбһ¶бһ„бҹ– $(z-3)^2 = (-4i)^2 \Rightarrow z^2 - 6z + 9 = -16$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $z^2 - 6z = -16 - 9 = -25$бҹ”
бһҡбһҖбһҳбһјбһҢбһ»бһӣбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = (1+i)(2-2i)(\sqrt{3}+i)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = |1+i| \times |2-2i| \times |\sqrt{3}+i|$бҹ”
  • $|1+i| = \sqrt{2}$, $|2-2i| = \sqrt{4+4} = 2\sqrt{2}$, $|\sqrt{3}+i| = \sqrt{3+1} = 2$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $|z| = \sqrt{2} \times 2\sqrt{2} \times 2 = 4 \times 2 = 8$бҹ”
бһҡбһҖбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -1 + i\sqrt{3}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ…бҹҶбһҺбһ»бһ…бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹў ($x < 0, y > 0$)бҹ”
  • $\tan\alpha = \left|\frac{\sqrt{3}}{-1}\right| = \sqrt{3} \Rightarrow \alpha = \frac{\pi}{3}$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ»бҹҶбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһӮбһә $\theta = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$бҹ”
бһҹбһҡбһҹбҹҒбһҡ $z = -4\left(\cos\frac{\pi}{6} - i\sin\frac{\pi}{6}\right)$ бһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = -4\cos\frac{\pi}{6} + 4i\sin\frac{\pi}{6}$бҹ” бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ (бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹў)бҹ”
  • бһҳбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹў бһӮбһә $\theta = \pi - \frac{\pi}{6} = \frac{5\pi}{6}$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһӮбһә $4\left(\cos\frac{5\pi}{6} + i\sin\frac{5\pi}{6}\right)$бҹ” бһ…бҹҶбһ–бҹ„бҹҮбһҮбһҳбҹ’бһҡбһҫбһҹбһҠбҹӮбһӣбһҹбҹ’бһҳбһҫбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ…бһҳбҹ’бһ”бһ„бһӮбһә $4\left(\cos\left(-\frac{5\pi}{6}\right) + i\sin\left(-\frac{5\pi}{6}\right)\right)$ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бһ„бҹ’бһңбһ·бһӣбһҹбһүбҹ’бһүбһ¶бҹ”
бһ”бһҫ $z = 2\left(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4}\right)$ бһӮбһҺбһ“бһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ“бҹғбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ $\overline{z}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҹбһүбҹ’бһүбһ¶бһҳбһ»бҹҶбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ‘бҹ…бһҮбһ¶бһ•бҹ’бһ‘бһ»бһҷбҹ– $\arg(\overline{z}) = -\arg(z) = -\frac{\pi}{4}$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮбһ”бһ¶бһ“бҹ– $2\left(\cos\left(-\frac{\pi}{4}\right) + i\sin\left(-\frac{\pi}{4}\right)\right)$ бһҠбҹӮбһӣбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бҹ”
бһҡбһҖбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ“бҹғбһ•бһӣбһӮбһ»бһҺ $z_1 \cdot z_2$ бһ”бһҫ $z_1 = -i$ бһ“бһ·бһ„ $z_2 = 1-i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\arg(z_1) = -\frac{\pi}{2}$ бһ“бһ·бһ„ $z_2 = 1-i$ бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹӨ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\arg(z_2) = -\frac{\pi}{4}$бҹ”
  • $\arg(z_1 \cdot z_2) = \arg(z_1) + \arg(z_2) = -\frac{\pi}{2} + \left(-\frac{\pi}{4}\right) = -\frac{3\pi}{4}$бҹ”
бһ”бҹ’бһҸбһјбһҡбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -2i$ бһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһўбһ·бһ…бһҹбҹ’бһ”бһјбһҺбһ„бҹӢбһҹбҹ’бһҷбҹӮбһӣбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҳбһјбһҢбһ»бһӣ $r = 2$ бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„ $\theta = -\frac{\pi}{2}$бҹ”
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһўбһҷбһӣбҹҗбһҡ $z = r e^{i\theta} = 2e^{-i\frac{\pi}{2}}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\left(\cos\frac{\pi}{10} + i\sin\frac{\pi}{10}\right)^{5}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһҠбһәбһҳбҹҗбһҡбҹ– $\cos\left(5 \times \frac{\pi}{10}\right) + i\sin\left(5 \times \frac{\pi}{10}\right) = \cos\frac{\pi}{2} + i\sin\frac{\pi}{2}$бҹ”
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $\cos\frac{\pi}{2} = 0$ бһ“бһ·бһ„ $\sin\frac{\pi}{2} = 1$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһә $i$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1+i)^{20}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶ $(1+i)^2 = 2i$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $(1+i)^{20} = ((1+i)^2)^{10} = (2i)^{10} = 2^{10} \times i^{10}$бҹ”
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $2^{10} = 1024$ бһ“бһ·бһ„ $i^{10} = i^2 = -1$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһӮбһә $-1024$бҹ”
бһ”бһҫ $z = \frac{1}{2} + i\frac{\sqrt{3}}{2}$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $1 + z + z^2$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = \cos\frac{\pi}{3} + i\sin\frac{\pi}{3}$бҹ”
  • $z^2 = \cos\frac{2\pi}{3} + i\sin\frac{2\pi}{3} = -\frac{1}{2} + i\frac{\sqrt{3}}{2}$бҹ”
  • бһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бҹ– $1 + \left(\frac{1}{2} + i\frac{\sqrt{3}}{2}\right) + \left(-\frac{1}{2} + i\frac{\sqrt{3}}{2}\right) = 1 + i\sqrt{3}$бҹ” (бһҖбҹӮбһҖбҹҶбһ бһ»бһҹбһҹбҹ’бһҳбһ¶бһ“бҹ– бһҮбһҳбҹ’бһҡбһҫбһҹбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„ $1+i\sqrt{3}$ бһҸбҹӮбһ”бһҫбһҮбһҳбҹ’бһҡбһҫбһҹбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӮбһә $1+i\sqrt{3}$)бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{(\cos 2\theta + i\sin 2\theta)^3}{\cos 4\theta + i\sin 4\theta}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ—бһ¶бһӮбһҷбһҖбҹ– $(\cos 2\theta + i\sin 2\theta)^3 = \cos 6\theta + i\sin 6\theta$бҹ”
  • бһ•бһӣбһ…бҹӮбһҖбҹ– $\cos(6\theta - 4\theta) + i\sin(6\theta - 4\theta) = \cos 2\theta + i\sin 2\theta$бҹ”
бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ $n$ бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷ $(1-i)^n$ бһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бһ“бһ·бһ„бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $1-i = \sqrt{2}\left(\cos\left(-\frac{\pi}{4}\right) + i\sin\left(-\frac{\pi}{4}\right)\right)$бҹ”
  • $(1-i)^n = (\sqrt{2})^n \left(\cos\left(-\frac{n\pi}{4}\right) + i\sin\left(-\frac{n\pi}{4}\right)\right)$бҹ”
  • бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһҸбҹ’бһҡбһјбһңбһҸбҹӮ $\sin\left(-\frac{n\pi}{4}\right) = 0$ бһ“бһ·бһ„ $\cos\left(-\frac{n\pi}{4}\right) = 1$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $-\frac{n\pi}{4} = -2k\pi \Rightarrow n = 8k$бҹ” аё•бһјбһ…бһ”бҹҶбһ•бһ»бһҸбһӮбһә $n=8$бҹ”
бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -3 - 4i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһ„бһ«бһҹбһҖбһ¶бһҡбҹҒ $x+yi$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $x^2-y^2 = -3$ бһ“бһ·бһ„ $2xy = -4 \Rightarrow xy = -2 < 0$ ($x,y$ бһҹбһүбҹ’бһүбһ¶бһ•бҹ’бһ‘бһ»бһҷбһӮбҹ’бһ“бһ¶)бҹ”
  • бһҳбһјбһҢбһ»бһӣ $x^2+y^2 = \sqrt{(-3)^2+(-4)^2} = 5$бҹ”
  • бһ”бһјбһҖбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбһёбһҖбһ¶бһҡбҹ– $2x^2 = 2 \Rightarrow x = \pm 1$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“ $y = \mp 2$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮбһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm(1-2i)$бҹ”
бһ”бһҫ $\omega$ бһҮбһ¶бһ«бһҹбһӮбһјбһ”бһҳбһ·бһ“бһ–бһ·бһҸбһ“бҹғ $1$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\frac{1}{1+\omega} + \frac{1}{1+\omega^2}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһҫбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ $1+\omega = -\omega^2$ бһ“бһ·бһ„ $1+\omega^2 = -\omega$бҹ”
  • бһҖбһ“бҹ’бһҹбҹ„бһҳбһ‘бҹ…бһҮбһ¶бҹ– $\frac{1}{-\omega^2} + \frac{1}{-\omega} = -\left(\frac{1+\omega}{\omega^2}\right) = -\left(\frac{-\omega^2}{\omega^2}\right) = 1$бҹ”
бһҡбһҖбһ«бһҹбһ‘бһё бҹӨ бһ“бҹғ $1$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡ $z^4 = 1 \Rightarrow z^4 - 1 = 0 \Rightarrow (z^2-1)(z^2+1) = 0$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $z^2 = 1 \Rightarrow z = \pm 1$ бһ¬ $z^2 = -1 \Rightarrow z = \pm i$бҹ”
бһӮбһҺбһ“бһ¶бһ•бһӣбһ”бһјбһҖбһ“бҹғбһ«бһҹбһ‘бһё бҹЈ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = 8i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҷбҹ„бһ„бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһ•бһӣбһ”бһјбһҖбһ«бһҹбһ‘бһё $n$ бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҺбһ¶бһҳбһҪбһҷ ($n \ge 2$) бһӮбһәбһҹбҹ’бһҳбһҫ $0$ бһҮбһ¶бһ“бһ·бһ…бҹ’бһ… бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҳбҹҒбһӮбһ»бһҺбһҸбһҪбһ‘бһё бҹў бһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $w^3 - 8i = 0$ бһӮбһәбһҹбһјбһ“бҹ’бһҷбҹ”
бһҡбһҖбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ“бҹғбһ«бһҹбһӮбһјбһ”бһ…бһҳбҹ’бһ”бһ„ ($k=0$) бһ“бҹғ $z = 8$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = 8(\cos 0 + i\sin 0)$бҹ”
  • бһ«бһҹбһ‘бһё бҹЈ бһӮбһә $w_k = \sqrt[3]{8}\left(\cos\frac{0+2k\pi}{3} + i\sin\frac{0+2k\pi}{3}\right)$бҹ”
  • бһ…бҹҶбһ–бҹ„бҹҮ $k=0$бҹ– $w_0 = 2(\cos 0 + i\sin 0) = 2$бҹ”
бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 4z + 13 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = (-2)^2 - 1(13) = 4 - 13 = -9 = 9i^2$бҹ”
  • бһ«бһҹбһҹбһҳбһёбһҖбһ¶бһҡбһӮбһә $z = 2 \pm \sqrt{9i^2} = 2 \pm 3i$бҹ”
бһҡбһҖбһ•бһӣбһӮбһ»бһҺбһ“бҹғбһ«бһҹбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $3z^2 + 5z + 12 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸ бһ•бһӣбһӮбһ»бһҺбһ«бһҹбһӮбһә $P = \frac{c}{a} = \frac{12}{3} = 4$бҹ”
бһҡбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһҠбһәбһҖбҹ’бһҡбҹҒбһ‘бһё бҹў бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“ $z = 1-i$ бһҮбһ¶бһ«бһҹбһҳбһҪбһҷ (бһҳбҹҒбһӮбһ»бһҺбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бһҫбһ«бһҹбһҳбһҪбһҷбһӮбһә $1-i$ бһ“бҹ„бҹҮбһ«бһҹбһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбһә $1+i$бҹ”
  • бһ•бһӣбһ”бһјбһҖбһ«бһҹ $S = (1-i) + (1+i) = 2$бҹ”
  • бһ•бһӣбһӮбһ»бһҺбһ«бһҹ $P = (1-i)(1+i) = 1^2 - i^2 = 2$бҹ”
  • бһҹбһҳбһёбһҖбһ¶бһҡбһӮбһә $z^2 - Sz + P = 0 \Rightarrow z^2 - 2z + 2 = 0$бҹ”
бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһҺбһ»бһ…бһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷ $|z + 4i| = |z - 4i|$ бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһҳбҹҒбһҠбҹ’бһҷбһ¶бһ‘бҹҗбһҡбһ“бҹғбһўбһ„бҹ’бһҖбһҸбҹӢбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ–бһёбһ…бҹҶбһҺбһ»бһ… $(0, -4)$ бһ‘бҹ… $(0, 4)$бҹ”
  • бһҳбҹҒбһҠбҹ’бһҷбһ¶бһ‘бҹҗбһҡбһ“бҹғбһ…бҹҶбһҺбһ»бһ…бһ–бһёбһҡбһ“бҹҒбҹҮбһӮбһәбһҒбҹ’бһҹбҹӮбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹҒбһҖ $y = 0$ бһҠбҹӮбһӣбһҮбһ¶бһўбҹҗбһҖбҹ’бһҹбһ–бһ·бһҸбҹ”
бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^4 - 16 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^4 - 16 = (z^2-4)(z^2+4) = 0$бҹ”
  • $z^2 = 4 \Rightarrow z = \pm 2$бҹ”
  • $z^2 = -4 \Rightarrow z = \pm 2i$бҹ” бһҹбҹҶбһҺбһ»бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷбһӮбһә $\{\pm 2, \pm 2i\}$бҹ”
