рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $i + 2i^2 + 3i^3 + 4i^4$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^1 = i$, $2i^2 = -2$, $3i^3 = -3i$, $4i^4 = 4$бҹ”
  • бһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бҹ– $(4 - 2) + (i - 3i) = 2 - 2i$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғ $k$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ бһ”бһҫ $z = 2 + ki$ бһ бһҫбһҷ $z^2$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһҹбһ»бһ‘бҹ’бһ’бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^2 = (2+ki)^2 = 4 - k^2 + 4ki$бҹ”
  • бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷ $z^2$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһҹбһ»бһ‘бҹ’бһ’ бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҹбһјбһ“бҹ’бһҷбҹ– $4 - k^2 = 0 \Rightarrow k^2 = 4 \Rightarrow k = \pm 2$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{5}{3-4i}$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ–бһёбһҮбһӮбһҺбһ·бһҸ $a+bi$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһӮбһ»бһҺбһ“бһ№бһ„бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ– $\frac{5(3+4i)}{(3-4i)(3+4i)} = \frac{15+20i}{9+16} = \frac{15+20i}{25}$бҹ”
  • бһҹбһҳбҹ’бһҡбһҪбһӣбһ“бһ№бһ„ бҹҘ бҹ– $\frac{3}{5} + \frac{4}{5}i$бҹ”
бһ”бһҫ $z = 1+2i$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҳбһјбһҢбһ»бһӣбһ“бҹғ $z^3$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһјбһҢбһ»бһӣбҹ– $|z^3| = |z|^3$бҹ”
  • $|z| = \sqrt{1^2+2^2} = \sqrt{5}$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $|z|^3 = (\sqrt{5})^3 = 5\sqrt{5}$бҹ”
бһҡбһҖбһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \frac{i}{1+i}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = \frac{i(1-i)}{(1+i)(1-i)} = \frac{i - i^2}{2} = \frac{1+i}{2} = \frac{1}{2} + \frac{1}{2}i$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһӮбһә $\frac{1}{2}$бҹ”
бһҡбһҖбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ…бһҳбҹ’бһ”бһ„бһ“бҹғ $z = -1 - i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $r = \sqrt{(-1)^2+(-1)^2} = \sqrt{2}$бҹ”
  • бһ…бҹҶбһҺбһ»бһ…бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹЈ ($x < 0, y < 0$) бһ“бһ¶бҹҶбһұбҹ’бһҷбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„ $\theta = -\pi + \frac{\pi}{4} = -\frac{3\pi}{4}$бҹ”
бһ”бһҫ $z = \cos\frac{\pi}{5} + i\sin\frac{\pi}{5}$ бһӮбһҺбһ“бһ¶бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһ“бҹғ $z^2 \cdot \overline{z}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\arg(z^2 \cdot \overline{z}) = \arg(z^2) + \arg(\overline{z}) = 2\arg(z) - \arg(z)$бҹ”
  • $\arg(z^2 \cdot \overline{z}) = \arg(z) = \frac{\pi}{5}$бҹ”
бһҡбһҖбһҳбһјбһҢбһ»бһӣбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \frac{1+i\sqrt{3}}{\cos\theta + i\sin\theta}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = \frac{|1+i\sqrt{3}|}{|\cos\theta+i\sin\theta|}$бҹ”
  • $|1+i\sqrt{3}| = \sqrt{1+3} = 2$ бһ“бһ·бһ„ $|\cos\theta+i\sin\theta| = 1$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $|z| = 2$бҹ”
бһҹбһҡбһҹбҹҒбһҡ $z = \sin\frac{\pi}{6} + i\cos\frac{\pi}{6}$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһҫбһҳбһ»бҹҶбһ”бҹҶбһ–бҹҒбһүбҹ– $\sin\frac{\pi}{6} = \cos\left(\frac{\pi}{2} - \frac{\pi}{6}\right) = \cos\frac{\pi}{3}$бҹ”
  • бһ“бһ·бһ„ $\cos\frac{\pi}{6} = \sin\left(\frac{\pi}{2} - \frac{\pi}{6}\right) = \sin\frac{\pi}{3}$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $z = \cos\frac{\pi}{3} + i\sin\frac{\pi}{3}$бҹ”
бһҡбһҖбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = e^{i\frac{4\pi}{3}}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҳбһ»бҹҶ $\frac{4\pi}{3}$ бһҳбһ·бһ“бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ…бһҳбҹ’бһ”бһ„ $(-\pi, \pi]$ бһ‘бҹҒбҹ”
  • бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҠбһҖбһ“бһ№бһ„ $2\pi$бҹ– $\frac{4\pi}{3} - 2\pi = -\frac{2\pi}{3}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1+i\sqrt{3})^6$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһӮбһә $2\left(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}\right)$бҹ”
  • бһҸбһ¶бһҳбһҠбһәбһҳбҹҗбһҡбҹ– $2^6\left(\cos\left(6 \times \frac{\pi}{3}\right) + i\sin\left(6 \times \frac{\pi}{3}\right)\right) = 64(\cos 2\pi + i\sin 2\pi) = 64(1+0) = 64$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{\cos 5\phi - i\sin 5\phi}{\cos 2\phi + i\sin 2\phi}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ—бһ¶бһӮбһҷбһҖбҹ– $\cos(-5\phi) + i\sin(-5\phi)$бҹ”
  • бһ•бһӣбһ…бҹӮбһҖбҹ– $\cos(-5\phi - 2\phi) + i\sin(-5\phi - 2\phi) = \cos(-7\phi) + i\sin(-7\phi)$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $i^{4n+3}$ бһ…бҹҶбһ–бҹ„бҹҮ $n \in \mathbb{N}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^{4n+3} = i^{4n} \cdot i^3 = (i^4)^n \cdot (-i) = 1^n \cdot (-i) = -i$бҹ”
бһ”бһҫ $z = \cos\alpha + i\sin\alpha$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\frac{1+z}{1-z}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһ–бһ¶бһҖбҹӢбһҖбһҺбҹ’бһҸбһ¶бһӣ бһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһӮбһә $i\cot\frac{\alpha}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1-i)^{10}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-i)^2 = -2i$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $(1-i)^{10} = (-2i)^5 = (-2)^5 \times i^5 = -32 \times i = -32i$бҹ”
бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғ $z = 2i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶бһҖбһ“бҹ’бһҹбҹ„бһҳбһҖбһ¶бһҡбҹҒбһ–бһ“бҹ’бһӣбһ¶бһҸбҹ– $(1+i)^2 = 1 + 2i - 1 = 2i$бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ бһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғ $2i$ бһӮбһә $\pm(1+i)$бҹ”
бһ”бһҫ $\omega$ бһҮбһ¶бһ«бһҹбһӮбһјбһ”бһҳбһ·бһ“бһ–бһ·бһҸбһ“бҹғ $1$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1+\omega^2)^3$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $1+\omega+\omega^2 = 0 \Rightarrow 1+\omega^2 = -\omega$бҹ”
  • $(1+\omega^2)^3 = (-\omega)^3 = -\omega^3 = -1$бҹ”
бһҡбһҖбһ•бһӣбһӮбһ»бһҺбһ“бҹғбһ«бһҹбһӮбһјбһ”бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғ $i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһӮбһ»бһҺбһ«бһҹбһ‘бһё $n$ бһ“бҹғ $z$ бһӮбһә $(-1)^{n-1}z$бҹ”
  • бһ…бҹҶбһ–бҹ„бҹҮ $n=3$ бһ“бһ·бһ„ $z=i$бҹ– бһ•бһӣбһӮбһ»бһҺбһӮбһә $(-1)^{3-1} \cdot i = (-1)^2 \cdot i = i$бҹ”
бһҸбһҫбһ«бһҹбһ‘бһё бҹӨ бһ“бҹғ $-16$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһҒбҹ’бһҹбҹӮбһҖбҹ„бһ„бһҺбһ¶бһҒбҹ’бһӣбҹҮбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһ‘бһё бҹӨ бһ“бҹғ $-16$ бһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһҹбҹ’бһҳбһҫбһӮбҹ’бһ“бһ¶бһӮбһә $R = \sqrt[4]{|-16|} = \sqrt[4]{16} = 2$бҹ”
  • бһ…бҹҶбһҺбһ»бһ…бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһҳбһ¶бһ“бһ…бһҳбҹ’бһ„бһ¶бһҷ бҹў бһҜбһҖбһҸбһ¶бһ–бһёбһӮбһӣбҹӢбһҸбһҳбҹ’бһҡбһ»бһҷ бһҠбһјбһ…бһ“бҹҒбҹҮбһңбһ¶бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһҡбһ„бҹ’бһңбһ„бҹӢбһ•бҹ’бһ…бһ·бһҸ $(0,0)$ бһҖбһ¶бҹҶбһҹбҹ’бһҳбһҫ $2$бҹ”
бһҡбһҖбһ«бһҹбһӮбһјбһ”бһ“бҹғбһҜбһҖбһҸбһ¶ ($1$) бһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһӮбһјбһ”бһҳбһ·бһ“бһ–бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹғ $1$ бһӮбһә $-\frac{1}{2} \pm i\frac{\sqrt{3}}{2}$бҹ”
  • бһ«бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ ($y > 0$) бһӮбһә $-\frac{1}{2} + i\frac{\sqrt{3}}{2}$бҹ”
бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + 2z + 2 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… Kelvinбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = 1^2 - 1(2) = -1 = i^2$бҹ”
  • бһ«бһҹбһӮбһә $z = -1 \pm i$бҹ”
бһ”бһҫ $z = 3 + i$ бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 6z + c = 0$ бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһӣбҹҒбһҒбһ“бҹғ $c$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһңбҹҖбһҸ бһ•бһӣбһӮбһ»бһҺбһ«бһҹ $P = z \cdot \overline{z} = c$ (бһ–бҹ’бһҡбҹ„бҹҮбһҳбҹҒбһӮбһ»бһҺбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸбһ“бһ¶бҹҶбһұбҹ’бһҷбһ«бһҹбһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбһә $3-i$)бҹ”
  • $c = (3+i)(3-i) = 3^2 - i^2 = 9 + 1 = 10$бҹ”
бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^3 - 1 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^3 - 1 = (z-1)(z^2+z+1) = 0$бҹ”
  • $z = 1$ бһ¬ $z^2+z+1=0 \Rightarrow z = \frac{-1 \pm i\sqrt{3}}{2}$бҹ”
бһҡбһҖбһҖбһ¶бҹҶ $R$ бһ“бҹғбһңбһ„бҹ’бһңбһ„бҹӢбһҠбҹӮбһӣбһҸбҹҶбһҺбһ¶бһ„бһҠбҹ„бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $|z - 3| = 5$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡ $|z - z_0| = R$ бһӮбһәбһҮбһ¶бһҡбһ„бҹ’бһңбһ„бҹӢбһ•бҹ’бһ…бһ·бһҸ $z_0$ бһҖбһ¶бҹҶ $R$бҹ”
  • бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ $z_0 = 3$ (бһ…бҹҶбһҺбһ»бһ… $(3,0)$) бһ“бһ·бһ„бһҖбһ¶бҹҶ $R = 5$бҹ”
бһҡбһҖбһ•бһӣбһ”бһјбһҖбһ«бһҹбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - (3+2i)z + 5 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸ бһ•бһӣбһ”бһјбһҖбһ«бһҹ $S = -\frac{b}{a}$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $S = -\frac{-(3+2i)}{1} = 3 + 2i$бҹ”
