រកតម្លៃនៃផលគុណឫសនៃសមីការកុំផ្លិច $3z^2 - 5z + (1+i) = 0$។ (៥ ពិន្ទុ)

  • តាមទ្រឹស្តីបទវៀតចំពោះសមីការដឺក្រេទី ២ ផលគុណឫស $P = \frac{c}{a}$។
  • នៅទីនេះ $c = 1+i$ និង $a = 3$ នាំឱ្យ $P = \frac{1+i}{3}$។
ដោះស្រាយសមីការ $z^2 + 4 = 0$ ក្នុងសំណុំកុំផ្លិច។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយ៖
  • សមីការអាចសរសេរជា $z^2 = -4$។
  • នាំឱ្យ $z = \pm \sqrt{-4} = \pm i\sqrt{4} = \pm 2i$។
កំណត់ទីតាំងធរណីមាត្រនៃចំណុច $M(z)$ ក្នុងប្លង់កុំផ្លិចដែលផ្ទៀងផ្ទាត់សមីការ $|z - (2+3i)| = 5$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយ៖
  • តាមនិយមន័យ សមីការទម្រង់ $|z - z_0| = R$ តំណាងឱ្យរង្វង់ដែលមានផ្ចិត $z_0$ និងកាំ $R$។
  • នៅទីនេះ $z_0 = 2+3i$ ត្រូវនឹងកូអរដោនេផ្ចិត $(2, 3)$ និងមានកាំ $R = 5$។
បើ $z = 1-i$ ជាឫសមួយនៃសមីការ $z^2 - 2z + c = 0$ (ដែល $c$ ជាចំនួនពិត) រកតម្លៃនៃ $c$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយ៖
  • ដោយសារ $c$ ជាចំនួនពិត ឫសមួយទៀតត្រូវតែជាចំនួនកុំផ្លិចឆ្លាស់គឺ $\overline{z} = 1+i$។
  • តាមទ្រឹស្តីបទវៀត ផលគុណឫសគឺ $P = \frac{c}{a} \Rightarrow z \cdot \overline{z} = \frac{c}{1}$។
  • គណនា $c = (1-i)(1+i) = 1^2 - i^2 = 1 + 1 = 2$។
រកសំណុំចម្លើយនៃសមីការ $|z| - z = 1 + 2i$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយ៖
  • តាង $z = x+yi$ នាំឱ្យ $|z| = \sqrt{x^2+y^2}$។
  • សមីការទៅជា៖ $\sqrt{x^2+y^2} - (x+yi) = 1 + 2i \Rightarrow (\sqrt{x^2+y^2} - x) - yi = 1 + 2i$។
  • ផ្ទឹមផ្នែកពិតនិងស្រមើស្រមៃ៖ $-y = 2 \Rightarrow y = -2$។
  • ជំនួស $y$ ក្នុងផ្នែកពិត៖ $\sqrt{x^2+(-2)^2} - x = 1 \Rightarrow \sqrt{x^2+4} = x + 1$។
  • លើកជាការេ៖ $x^2 + 4 = x^2 + 2x + 1 \Rightarrow 2x = 3 \Rightarrow x = \frac{3}{2}$។
  • ដូចនេះចម្លើយគឺ $z = \frac{3}{2} - 2i$។
