F

рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһ”бһҫ $z = \frac{1+i}{1-i}$ бһҸбһҳбҹ’бһӣбҹғбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ–бһёбһҮбһӮбһҺбһ·бһҸбһҡбһ”бһҹбҹӢ $z$ бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\frac{1+i}{1-i} = \frac{(1+i)^2}{1^2-(-1)^2} = \frac{2i}{2} = i$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(2-3i) + (-5+i) - (1-2i)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбҹ– $2 - 5 - 1 = -4$бҹ” бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбҹғбҹ– $-3i + i - (-2i) = 0$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһә $-4$бҹ”
бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z$ бһҠбҹӮбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһёбһҖбһ¶бһҡ $2z + 3\overline{z} = 5 - i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһ„ $z=x+yi \Rightarrow 2(x+yi)+3(x-yi) = 5x - yi = 5 - i$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷ $5x = 5 \Rightarrow x = 1$ бһ“бһ·бһ„ $-y = -1 \Rightarrow y = 1$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $z = 1-i$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $i^{15} - i^{13}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^{15} = i^3 = -i$ бһ“бһ·бһ„ $i^{13} = i^1 = i$бҹ”
  • $(-i) - (i) = -2i$бҹ”
бһ”бһҫ $z = (2+i)(1-2i)$ бһҸбһҫбһҳбһјбһҢбһ»бһӣбһ“бҹғ $z$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = |2+i| \cdot |1-2i| = \sqrt{2^2+1^2} \cdot \sqrt{1^2+(-2)^2} = \sqrt{5} \cdot \sqrt{5} = 5$бҹ”
бһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \sqrt{3} - i$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = 2$бҹ” бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹӨ ($\text{Re}>0, \text{Im}<0$) бһ“бһ¶бҹҶбһұбҹ’бһҷ $\theta = -\frac{\pi}{6}$бҹ”
бһҡбһҖбһҳбһјбһҢбһ»бһӣ бһ“бһ·бһ„бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ“бҹғ $z = -2 + 2i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $r = \sqrt{(-2)^2+2^2} = \sqrt{8} = 2\sqrt{2}$бҹ” бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹў бһ“бһ¶бҹҶбһұбҹ’бһҷ $\theta = \pi - \frac{\pi}{4} = \frac{3\pi}{4}$бҹ”
бһ”бһҫ $z = \cos \alpha - i\sin \alpha$ бһҸбһҫбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҡбһ”бһҹбҹӢбһңбһ¶бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ– $\cos(-\alpha) = \cos \alpha$ бһ“бһ·бһ„ $\sin(-\alpha) = -\sin \alpha$бҹ”
бһӮбһҺбһ“бһ¶бһ•бһӣбһӮбһ»бһҺ $z_1 \cdot z_2$ бһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡ бһ”бһҫ $z_1 = \sqrt{2}(\cos \frac{\pi}{3} + i\sin \frac{\pi}{3})$ бһ“бһ·бһ„ $z_2 = \sqrt{2}(\cos \frac{\pi}{6} + i\sin \frac{\pi}{6})$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $r = \sqrt{2} \times \sqrt{2} = 2$ бһ“бһ·бһ„ $\theta = \frac{\pi}{3} + \frac{\pi}{6} = \frac{\pi}{2}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{6(\cos \frac{5\pi}{3} + i\sin \frac{5\pi}{3})}{2(\cos \frac{\pi}{3} + i\sin \frac{\pi}{3})}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ•бһӣбһ…бҹӮбһҖбҹ– $\frac{6}{2} = 3$ бһ“бһ·бһ„бһҳбһ»бҹҶ $\frac{5\pi}{3} - \frac{\pi}{3} = \frac{4\pi}{3}$бҹ”
  • $3(\cos \frac{4\pi}{3} + i\sin \frac{4\pi}{3}) = 3(-\frac{1}{2} - i\frac{\sqrt{3}}{2})$... бһҹбҹҶбһҺбһҪбһҡбһ“бҹҒбҹҮбһҹбһ»бҹҶбһҖбҹӮбһ…бһҳбҹ’бһӣбһҫбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡ $3(\cos \frac{4\pi}{3} + i\sin \frac{4\pi}{3})$ бһңбһ·бһүбһ”бһҫбһӮбһ·бһҸбһҸбһ¶бһҳбһҮбҹҶбһ бһ¶бһ“бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ”бһҫбһӮбһ·бһҸбһҸбһҳбҹ’бһӣбҹғ $\cos \frac{4\pi}{3} = -0.5$бҹ” бһҹбһ»бҹҶбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһҳбҹ’бһҡбһҫбһҹ C бһұбҹ’бһҷбһҸбҹ’бһҡбһјбһңбһӮбһәбҹ– $3(\cos \frac{4\pi}{3} + i\sin \frac{4\pi}{3})$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1+i)^{10}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1+i)^{10} = [(1+i)^2]^5 = (2i)^5 = 32i^5 = 32i$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(\cos \frac{\pi}{4} - i\sin \frac{\pi}{4})^8$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $[\cos(-\frac{\pi}{4}) + i\sin(-\frac{\pi}{4})]^8 = \cos(-2\pi) + i\sin(-2\pi) = 1$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{(\cos 2\theta + i\sin 2\theta)^3}{\cos 3\theta + i\sin 3\theta}$ бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһҠбһәбһҳбҹҗбһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ—бһ¶бһӮбһҷбһҖбҹ– $\cos 6\theta + i\sin 6\theta$бҹ”
  • бһ•бһӣбһ…бҹӮбһҖбҹ– $\cos(6\theta - 3\theta) + i\sin(6\theta - 3\theta) = \cos 3\theta + i\sin 3\theta$бҹ”
бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(-\frac{\sqrt{3}}{2} + \frac{1}{2}i)^{12}$ бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҳбһ»бҹҶбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһӮбһә $\frac{5\pi}{6}$бҹ” бһӣбһҫбһҖбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһӮбһ»бһҺ бҹЎбҹўбҹ– $\cos(12 \cdot \frac{5\pi}{6}) + i\sin(12 \cdot \frac{5\pi}{6}) = \cos(10\pi) + i\sin(10\pi) = 1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $i^{-35}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^{-35} = \frac{1}{i^{35}} = \frac{1}{i^3} = \frac{1}{-i} = \frac{i}{-i^2} = i$бҹ”
бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -7 + 24i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(3+4i)^2 = 9 + 24i - 16 = -7 + 24i$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm (3+4i)$бҹ”
бһҸбһҫбһ«бһҹбһӮбһјбһ”бһ“бҹғ $i$ бһҳбһ¶бһ“бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһҸбһјбһ…бһҮбһ¶бһ„бһӮбҹҒ (бһ…бһҳбҹ’бһ”бһ„) бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i = \cos \frac{\pi}{2} + i\sin \frac{\pi}{2}$бҹ” бһ«бһҹбһ‘бһё бҹЈ бһ‘бһёбһҳбһҪбһҷ ($k=0$) бһҳбһ¶бһ“бһҳбһ»бҹҶ $\theta_0 = \frac{\pi/2}{3} = \frac{\pi}{6}$бҹ”
бһ•бһӣбһ”бһјбһҖбһ“бҹғбһ«бһҹбһ‘бһё бҹӨ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғ $1$ бһӮбһәбһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһ‘бһё бҹӨ бһ“бҹғ $1$ бһӮбһә $1, -1, i, -i$бҹ” бһ•бһӣбһ”бһјбһҖбһҡбһ”бһҹбҹӢбһңбһ¶бһӮбһә $1 + (-1) + i + (-i) = 0$бҹ”
бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғ $-1 + i\sqrt{3}$ бһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = 2(\cos \frac{2\pi}{3} + i\sin \frac{2\pi}{3})$бҹ” бһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm \sqrt{2}(\cos \frac{2\pi/3}{2} + i\sin \frac{2\pi/3}{2}) = \pm \sqrt{2}(\cos \frac{\pi}{3} + i\sin \frac{\pi}{3})$бҹ”
бһҸбһҫбһ…бҹҶбһҺбһ»бһ…бһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһ«бһҹбһ‘бһё бҹЈ бһ“бҹғ $1$ бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бһҮбһ¶бһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһўбҹ’бһңбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһ‘бһё $n$ бһ“бҹғ $1$ бһҸбҹӮбһ„бһҸбҹӮбһ…бҹӮбһҖбһҡбһ„бҹ’бһңбһ„бҹӢбһҜбһҖбһҸбһ¶бһҮбһ¶ $n$ бһ…бҹҶбһҺбҹӮбһҖбһҹбҹ’бһҳбһҫбҹ—бһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһ“бҹҒбҹҮбһ…бҹҶбһҺбһ»бһ…бһҸбҹҶбһҺбһ¶бһ„бһҡбһ”бһҹбҹӢбһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ¶бһ“бһҮбһ¶бһ–бһ бһ»бһҖбҹ„бһҺбһ“бһ·бһҷбҹҗбһҸ (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ $n=3$ бһӮбһәбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҹбһҳбҹҗбһ„бҹ’бһҹ)бҹ”
бһҡбһҖбһҹбҹҶбһҺбһ»бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 4z + 13 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = (-2)^2 - 1(13) = 4 - 13 = -9 = (3i)^2 \Rightarrow z = 2 \pm 3i$бҹ”
бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - \sqrt{2}z + 1 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta = (-\sqrt{2})^2 - 4(1)(1) = 2 - 4 = -2 = (i\sqrt{2})^2$бҹ”
  • $z = \frac{\sqrt{2} \pm i\sqrt{2}}{2}$бҹ”
бһ”бһҫ $z^3 - 1 = 0$ бһҸбһҫбһҹбҹҶбһҺбһ»бҹҶбһ«бһҹбһҡбһ”бһҹбҹӢбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^3-1 = (z-1)(z^2+z+1) = 0 \Rightarrow z=1$ бһ¬ $z = \frac{-1 \pm i\sqrt{3}}{2}$бҹ”
бһҡбһҖбһҸбһҪбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһ·бһҸбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z + \frac{1}{z} = 1$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһӮбһ»бһҺбһ“бһ№бһ„ $z$ бһ‘бһ¶бҹҶбһ„бһҹбһ„бһҒбһ¶бһ„бҹ– $z^2 - z + 1 = 0 \Rightarrow \Delta = (-1)^2 - 4(1)(1) = -3 = (i\sqrt{3})^2$бҹ”
  • $z = \frac{1 \pm i\sqrt{3}}{2}$бҹ”
бһҸбһҫбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 2x \cdot z + (x^2+y^2) = 0$ (бһҠбҹӮбһӣ $x, y \in \mathbb{R}$) бһҳбһ¶бһ“бһ«бһҹбһҮбһ¶бһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = (-x)^2 - (1)(x^2+y^2) = x^2 - x^2 - y^2 = -y^2 = (yi)^2$бҹ”
  • $z = x \pm yi$бҹ”
