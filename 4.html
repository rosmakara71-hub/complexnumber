рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $i^6 + i^{16} + i^{26} + i^{36}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^6 = i^2 = -1$ , $i^{16} = 1$ , $i^{26} = i^2 = -1$ , $i^{36} = 1$бҹ”
  • бһ•бһӣбһ”бһјбһҖбҹ– $(-1) + 1 + (-1) + 1 = 0$бҹ”
бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһ”бһүбҹ’бһҡбҹ’бһ…бһ¶бһҹбҹӢ (Multiplicative Inverse) бһ“бҹғ $z = 1 + i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^{-1} = \frac{1}{1+i} = \frac{1-i}{1^2+1^2} = \frac{1-i}{2} = \frac{1}{2} - \frac{1}{2}i$бҹ”
бһ”бһҫ $z_1 = 3 + 4i$ бһ“бһ·бһ„ $z_2 = 1 - i$ бһҸбһҫбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһ“бҹғ $z_1 \cdot z_2$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z_1 \cdot z_2 = (3+4i)(1-i) = 3 - 3i + 4i - 4i^2 = 3 + i + 4 = 7 + i$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһӮбһә $7$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{i^{100} + i^{102}}{i^{101}}$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбһ¶бһҳбһүбҹ’бһүбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^{100} = 1$ бһ“бһ·бһ„ $i^{102} = -1$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ—бһ¶бһӮбһҷбһҖ $1 + (-1) = 0$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбҹ’бһҳбһҫ $0$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸ $x$ бһ“бһ·бһ„ $y$ бһҠбҹӮбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһ—бһ¶бһ– $(x+y) + (x-y)i = 6 + 2i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбһёбһҖбһ¶бһҡбҹ– $x + y = 6$ бһ“бһ·бһ„ $x - y = 2$бҹ”
  • бһ”бһјбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ…бһјбһӣбһӮбҹ’бһ“бһ¶бҹ– $2x = 8 \Rightarrow x = 4$бҹ” бһ“бһ¶бҹҶбһұбҹ’бһҷ $y = 6 - 4 = 2$бҹ”
бһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -3i$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ…бһҳбҹ’бһ”бһ„ (бһҠбҹӮбһӣбһҳбһ»бҹҶбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бһё $-\pi$ бһҠбһӣбҹӢ $\pi$)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = 3$бҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡ $z$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһӮбһә $-\frac{\pi}{2}$бҹ”
бһҡбһҖбһҳбһјбһҢбһ»бһӣбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \frac{1+i\sqrt{3}}{1-i}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = \frac{|1+i\sqrt{3}|}{|1-i|} = \frac{\sqrt{1^2+(\sqrt{3})^2}}{\sqrt{1^2+(-1)^2}} = \frac{2}{\sqrt{2}} = \sqrt{2}$бҹ”
бһ”бһҫ $z = -2(\cos \frac{\pi}{4} + i\sin \frac{\pi}{4})$ бһҸбһҫбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҡбһ”бһҹбҹӢбһңбһ¶ (бһҠбҹӮбһӣбһҳбһјбһҢбһ»бһӣбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“) бһӮбһәбһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $-1 = \cos \pi + i\sin \pi$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $z = 2(\cos \pi + i\sin \pi)(\cos \frac{\pi}{4} + i\sin \frac{\pi}{4})$бҹ”
  • бһ”бһјбһҖбһҳбһ»бҹҶбһ…бһјбһӣбһӮбҹ’бһ“бһ¶бҹ– $\pi + \frac{\pi}{4} = \frac{5\pi}{4}$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $z = 2(\cos \frac{5\pi}{4} + i\sin \frac{5\pi}{4})$бҹ”
бһҡбһҖбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \cos \frac{4\pi}{3} - i\sin \frac{4\pi}{3}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = \cos(-\frac{4\pi}{3}) + i\sin(-\frac{4\pi}{3})$бҹ” бһ”бһ“бҹ’бһҗбҹӮбһҳ $2\pi$ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ…бһ“бҹ’бһӣбҹ„бҹҮбһ…бһҳбҹ’бһ”бһ„бҹ– $-\frac{4\pi}{3} + 2\pi = \frac{2\pi}{3}$бҹ”
бһӮбһҺбһ“бһ¶бһ•бһӣбһ…бҹӮбһҖ $\frac{z_1}{z_2}$ бһ”бһҫ $z_1 = 8(\cos \frac{\pi}{3} + i\sin \frac{\pi}{3})$ бһ“бһ·бһ„ $z_2 = 2(\cos \frac{\pi}{2} + i\sin \frac{\pi}{2})$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\frac{8}{2} = 4$ бһ“бһ·бһ„бһҳбһ»бҹҶбһҹбҹ’бһҳбһҫбһ“бһ№бһ„ $\frac{\pi}{3} - \frac{\pi}{2} = -\frac{\pi}{6}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(\frac{1}{\sqrt{2}} + i\frac{1}{\sqrt{2}})^{100}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһӮбһә $\cos \frac{\pi}{4} + i\sin \frac{\pi}{4}$бҹ”
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһҠбһәбһҳбҹҗбһҡбҹ– $\cos(100 \cdot \frac{\pi}{4}) + i\sin(100 \cdot \frac{\pi}{4}) = \cos(25\pi) + i\sin(25\pi) = \cos \pi + i\sin \pi = -1$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1 - i)^8$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-i)^2 = -2i \Rightarrow (-2i)^4 = 16i^4 = 16$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{(\cos \theta - i\sin \theta)^4}{(\cos \theta + i\sin \theta)^3}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ—бһ¶бһӮбһҷбһҖбҹ– $(\cos(-\theta) + i\sin(-\theta))^4 = \cos(-4\theta) + i\sin(-4\theta)$бҹ”
  • бһ—бһ¶бһӮбһ”бҹӮбһ„бҹ– $\cos 3\theta + i\sin 3\theta$бҹ”
  • бһ•бһӣбһ…бҹӮбһҖбҹ– $\cos(-4\theta - 3\theta) + i\sin(-4\theta - 3\theta) = \cos(-7\theta) + i\sin(-7\theta) = \cos 7\theta - i\sin 7\theta$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(\sqrt{3} - i)^6$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһӮбһә $2(\cos(-\frac{\pi}{6}) + i\sin(-\frac{\pi}{6}))$бҹ”
  • бһӣбһҫбһҖбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһӮбһ»бһҺ бҹҰбҹ– $2^6(\cos(-\pi) + i\sin(-\pi)) = 64(-1) = -64$бҹ”
бһ”бһҫ $z = \cos \frac{\pi}{3} + i\sin \frac{\pi}{3}$ бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $1 + z + z^2 + z^3 + z^4 + z^5$ бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ“бҹҒбҹҮбһҮбһ¶бһ•бһӣбһ”бһјбһҖбһ’бһҡбһҺбһёбһҳбһ¶бһҸбҹ’бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“ $q = z$ бһ“бһ·бһ„ $z^6 = \cos(2\pi)+i\sin(2\pi) = 1$бҹ”
  • $S_6 = \frac{1-z^6}{1-z} = \frac{1-1}{1-z} = 0$бҹ”
бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -3 - 4i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-2i)^2 = 1 - 4i + 4i^2 = 1 - 4i - 4 = -3 - 4i$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm (1-2i)$бҹ”
бһҸбһҫбһ«бһҹбһӮбһјбһ”бһ“бҹғ $-8i$ бһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҳбһјбһҢбһ»бһӣбһ“бҹғ $-8i$ бһӮбһә $r = 8$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһҳбһјбһҢбһ»бһӣбһ“бҹғбһ«бһҹбһ‘бһё бҹЈ бһӮбһә $R = \sqrt[3]{8} = 2$ бһҹбҹ’бһҳбһҫбҹ—бһӮбҹ’бһ“бһ¶бһӮбҹ’бһҡбһ”бҹӢбһ«бһҹбҹ”
бһҡбһҖбһ«бһҹбһ‘бһё бҹӨ бһ“бҹғ $-16$ бһҠбҹӮбһӣбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹЎ бһ“бҹғбһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $-16 = 16(\cos \pi + i\sin \pi)$бҹ” бһ«бһҹбһ‘бһё бҹӨ бһ‘бһёбһҳбһҪбһҷ ($k=0$) бһӮбһә $2(\cos \frac{\pi}{4} + i\sin \frac{\pi}{4}) = 2(\frac{\sqrt{2}}{2} + i\frac{\sqrt{2}}{2}) = \sqrt{2} + i\sqrt{2}$бҹ”
бһ”бһҫ $z_0, z_1, z_2$ бһҮбһ¶бһ«бһҹбһӮбһјбһ”бһ“бҹғ $1$ бһ“бҹ„бҹҮбһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $z_0 \cdot z_1 \cdot z_2$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбһёбһҖбһ¶бһҡ $z^3 - 1 = 0$ бһ•бһӣбһӮбһ»бһҺбһ«бһҹбһӮбһә $(-1)^3 \cdot (-1) = 1$бҹ”
бһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғ $i$ бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(\frac{1+i}{\sqrt{2}})^2 = \frac{1+2i-1}{2} = \frac{2i}{2} = i$бҹ”
бһҡбһҖбһ«бһҹбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 6z + 10 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = (-3)^2 - 1(10) = 9 - 10 = -1 = i^2 \Rightarrow z = 3 \pm i$бҹ”
бһ”бһҫ $2-i$ бһҮбһ¶бһ«бһҹбһҳбһҪбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + az + b = 0$ (бһҠбҹӮбһӣ $a, b \in \mathbb{R}$) бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $a$ бһ“бһ·бһ„ $b$ бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбһә $2+i$бҹ” бһ•бһӣбһ”бһјбһҖбһ«бһҹ $S = 4 \Rightarrow a = -S = -4$бҹ”
  • бһ•бһӣбһӮбһ»бһҺбһ«бһҹ $P = 2^2+1^2 = 5 \Rightarrow b = P = 5$бҹ”
бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^4 - 1 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(z^2-1)(z^2+1) = 0 \Rightarrow z^2=1$ бһ¬ $z^2=-1$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $z = \pm 1, \pm i$бҹ”
бһҡбһҖбһҹбҹҶбһҺбһ»бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 4iz - 4 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһўбһ¶бһ…бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҖбһ¶бһҡбҹҒбҹ– $(z - 2i)^2 = 0 \Rightarrow z = 2i$ (бһ«бһҹбһҢбһ»бһ”)бҹ”
бһҸбһҫбһ•бһӣбһ”бһјбһҖбһ“бҹғбһ«бһҹбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - (3+2i)z + 5 = 0$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸ бһ•бһӣбһ”бһјбһҖбһ«бһҹ $S = -\frac{b}{a} = -(-(3+2i)) = 3+2i$бҹ”
