рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бҹўбҹҰ. бһӮбһҺбһ“бһ¶бһ•бһӣбһӮбһ»бһҺ $(2 + 3i)(2 - 3i)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(2+3i)(2-3i) = 2^2 - (3i)^2 = 4 - 9i^2 = 4 + 9 = 13$бҹ”
бҹўбҹ§. бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{5}{i}$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ–бһёбһҮбһӮбһҺбһ·бһҸбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһӮбһ»бһҺбһ“бһ№бһ„ $i$ бһ‘бһ¶бҹҶбһ„бһ—бһ¶бһӮбһҷбһҖбһ“бһ·бһ„бһ—бһ¶бһӮбһ”бҹӮбһ„бҹ– $\frac{5i}{i^2} = \frac{5i}{-1} = -5i$бҹ”
бҹўбҹЁ. бһҡбһҖбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸ бһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбҹғбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = \frac{3-i}{2}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ•бҹ’бһҠбһ¶бһ…бҹӢбһ—бһ¶бһӮбһ”бҹӮбһ„бҹ– $z = \frac{3}{2} - \frac{1}{2}i$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $\text{Re}(z)=\frac{3}{2}$ бһ“бһ·бһ„ $\text{Im}(z)=-\frac{1}{2}$бҹ”
бҹўбҹ©. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1+i)^3$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1+i)^3 = (1+i)^2(1+i) = 2i(1+i) = 2i + 2i^2 = -2 + 2i$бҹ”
бҹЈбҹ . бһҡбһҖбһҸбһҳбҹ’бһӣбҹғ $x$ бһ“бһ·бһ„ $y$ бһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸбһҠбҹӮбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһҳбһ—бһ¶бһ– $x + yi = 4 - i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ•бҹ’бһ‘бһ№бһҳбһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸбһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҸ бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбҹғбһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбҹғбҹ– $x = 4$ бһ“бһ·бһ„ $y = -1$бҹ”
бҹЈбҹЎ. бһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -2$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = 2$, бһҠбҹ„бһҷбһҹбһ¶бһҡ $z$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹбһ–бһ·бһҸбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷ $\theta = \pi$бҹ”
бҹЈбҹў. бһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -i$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = 1$, бһҳбһ»бҹҶбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһӮбһә $-\frac{\pi}{2}$ (бһ¬ $\frac{3\pi}{2}$)бҹ”
бҹЈбҹЈ. бһҹбһҡбһҹбҹҒбһҡ $z = -1 + i\sqrt{3}$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = \sqrt{(-1)^2+(\sqrt{3})^2} = 2$бҹ” бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹў бһ“бһ¶бҹҶбһұбҹ’бһҷ $\theta = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$бҹ”
бҹЈбҹӨ. бһ”бһҫ $z = 2(\cos \frac{\pi}{6} + i\sin \frac{\pi}{6})$ бһҸбһҫбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ–бһёбһҮбһӮбһҺбһ·бһҸбһҡбһ”бһҹбҹӢ $z$ бһӮбһәбһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = 2(\frac{\sqrt{3}}{2} + i \cdot \frac{1}{2}) = \sqrt{3} + i$бҹ”
бҹЈбҹҘ. бһӮбһҺбһ“бһ¶бһ•бһӣбһ…бҹӮбһҖ $\frac{z_1}{z_2}$ бһ”бһҫ $z_1 = 4(\cos \frac{\pi}{2} + i\sin \frac{\pi}{2})$ бһ“бһ·бһ„ $z_2 = 2(\cos \frac{\pi}{6} + i\sin \frac{\pi}{6})$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһ…бҹӮбһҖбҹ– бһ…бҹӮбһҖбһҳбһјбһҢбһ»бһӣ бһҠбһҖбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢ $\frac{4}{2} = 2$ бһ“бһ·бһ„ $\frac{\pi}{2} - \frac{\pi}{6} = \frac{\pi}{3}$бҹ”
бҹЈбҹҰ. бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһҠбһәбһҳбҹҗбһҡ (De Moivre's Theorem) бһҖбһ“бҹ’бһҹбҹ„бһҳ $(\cos \theta + i\sin \theta)^{-1}$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(\cos \theta + i\sin \theta)^{-1} = \cos(-\theta) + i\sin(-\theta) = \cos \theta - i\sin \theta$бҹ”
бҹЈбҹ§. бһӮбһҺбһ“бһ¶ $[\sqrt{2}(\cos \frac{\pi}{4} + i\sin \frac{\pi}{4})]^4$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(\sqrt{2})^4[\cos(4 \cdot \frac{\pi}{4}) + i\sin(4 \cdot \frac{\pi}{4})] = 4(\cos \pi + i\sin \pi) = 4(-1 + 0) = -4$бҹ”
бҹЈбҹЁ. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(-\frac{1}{2} + i\frac{\sqrt{3}}{2})^3$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһӮбһә $\cos \frac{2\pi}{3} + i\sin \frac{2\pi}{3}$бҹ” бһӣбһҫбһҖбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһӮбһ»бһҺ бҹЈ бһӮбһә $\cos(2\pi) + i\sin(2\pi) = 1$бҹ”
бҹЈбҹ©. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғ $i^{50}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $50 = 4 \times 12 + 2$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $i^{50} = i^2 = -1$бҹ”
бҹӨбҹ . бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $(1-i)^6$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-i)^6 = [(1-i)^2]^3 = (-2i)^3 = -8i^3 = -8(-i) = 8i$бҹ”
бҹӨбҹЎ. бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғ $-9$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^2 = -9 = 9i^2 \Rightarrow z = \pm 3i$бҹ”
бҹӨбҹў. бһҸбһҫбһ…бҹҶбһ“бһҪбһ“бһ«бһҹбһ‘бһё бҹҘ бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = 1+i$ бһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… бһӮбҹ’бһҡбһ”бҹӢбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҳбһ·бһ“бһҹбһјбһ“бҹ’бһҷбһҸбҹӮбһ„бһҳбһ¶бһ“бһ«бһҹбһ‘бһё $n$ бһ…бҹҶбһ“бһҪбһ“ $n$ бһ•бҹ’бһҹбҹҒбһ„бһӮбҹ’бһ“бһ¶бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бҹ”
бҹӨбҹЈ. бһ«бһҹбһӮбһјбһ” (бһ«бһҹбһ‘бһё бҹЈ) бһ“бҹғ $-1$ бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(-1)^3 = -1$ бһҠбһјбһ…бһ“бҹҒбҹҮ $-1$ бһӮбһәбһҮбһ¶бһ«бһҹбһӮбһјбһ”бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҳбһҪбһҷбһҡбһ”бһҹбҹӢ $-1$бҹ”
бҹӨбҹӨ. бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -2i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-i)^2 = 1 - 2i + i^2 = -2i$ бһҠбһјбһ…бһ“бҹҒбҹҮбһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm (1-i)$бҹ”
бҹӨбҹҘ. бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһӣбһ„бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… бһ«бһҹбһ‘бһё $n$ бһ‘бһ¶бҹҶбһ„бһЎбһ¶бһҷбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҳбһҪбһҷ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһ‘бһё $n$ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһҹбҹ’бһҳбһҫбҹ—бһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһ“бҹҒбҹҮбһңбһ¶бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһҡбһ„бҹ’бһңбһ„бҹӢбһҸбҹӮбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бҹҶ $R = \sqrt[n]{r}$бҹ”
бҹӨбҹҰ. бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + 4 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^2 = -4 = 4i^2 \Rightarrow z = \pm 2i$бҹ”
бҹӨбҹ§. бһҡбһҖбһ«бһҹбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 + 2z + 2 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = 1^2 - 1(2) = -1 = i^2 \Rightarrow z = -1 \pm i$бҹ”
бҹӨбҹЁ. бһ”бһҫ $z = 1+i$ бһҮбһ¶бһ«бһҹбһҳбһҪбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡбһҠбһәбһҖбҹ’бһҡбҹҒбһ‘бһёбҹў бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбҹҒбһӮбһ»бһҺбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ бһҸбһҫбһ«бһҹбһҳбһҪбһҷбһ‘бҹҖбһҸбһӮбһәбһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бһҫбһҳбҹҒбһӮбһ»бһҺбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ бһ«бһҹбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҸбҹӮбһ„бһҸбҹӮбһҖбһҫбһҸбһЎбһҫбһ„бһҮбһ¶бһӮбһјбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһ“бһ¶ ($\overline{z} = 1-i$)бҹ”
бҹӨбҹ©. бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 2z + 5 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = (-1)^2 - 1(5) = -4 = (2i)^2 \Rightarrow z = 1 \pm 2i$бҹ”
бҹҘбҹ . бһҡбһҖбһҸбһҪбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһ–бһ·бһҮбһӮбһҺбһ·бһҸбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ–бҹҒбһүбһҹбһҳбһёбһҖбһ¶бһҡбҹ– $z^2 - 2iz - 1 = 0$ бһҳбһ¶бһ“бһ«бһҹбһҢбһ»бһ”бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҹбһҳбһёбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҮбһ¶ $(z-i)^2 = 0 \Rightarrow z = i$ (бһ«бһҹбһҢбһ»бһ”)бҹ”
