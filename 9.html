рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бҹўбҹ бҹЎ. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $i + i^2 + i^3 + \dots + i^{20}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ•бһӣбһ”бһјбһҖбһҹбҹ’бһңбҹҗбһҷбһӮбһ»бһҺбһ”бһҪбһ“бһҸбһҪбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһ“бҹғ $i$ бһӮбһәбһҹбҹ’бһҳбһҫ $0$ бһҮбһ¶бһ“бһ·бһ…бҹ’бһ… (бһ–бҹ’бһҡбҹ„бҹҮ $i - 1 - i + 1 = 0$)бҹ”
  • бһ…бһ¶бһ”бҹӢбһ–бһё $i^1$ бһҠбһӣбҹӢ $i^{20}$ бһҳбһ¶бһ“ бҹўбҹ  бһҸбһҪ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҮбһ¶ бҹҘ бһҖбҹ’бһҡбһ»бһҳ (бһ…бҹӮбһҖбһҠбһ¶бһ…бҹӢбһ“бһ№бһ„ бҹӨ) бһҠбһјбһ…бһ“бҹҒбҹҮбһ•бһӣбһ”бһјбһҖбһҹбһҡбһ»бһ”бһӮбһә $0$бҹ”
бҹўбҹ бҹў. бһҡбһҖбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһ…бҹ’бһҡбһ¶бһҹ бһ“бҹғ $z = 3 + 4i$ (Multiplicative inverse $z^{-1}$)бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^{-1} = \frac{1}{z} = \frac{\overline{z}}{|z|^2}$бҹ”
  • бһҷбһҫбһ„бһҳбһ¶бһ“ $|z|^2 = 3^2 + 4^2 = 25$ бһ“бһ·бһ„ $\overline{z} = 3 - 4i$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $z^{-1} = \frac{3-4i}{25} = \frac{3}{25} - \frac{4}{25}i$бҹ”
бҹўбҹ бҹЈ. бһ”бһҫ $z = x + yi$ бһ бһҫбһҷ $|z - i| = |z + i|$ бһҸбһҫбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„ $x$ бһ“бһ·бһ„ $y$ бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|x + (y-1)i|^2 = |x + (y+1)i|^2 \Rightarrow x^2 + (y-1)^2 = x^2 + (y+1)^2$бҹ”
  • бһ–бһ“бҹ’бһӣбһ¶бһҸбҹ– $y^2 - 2y + 1 = y^2 + 2y + 1 \Rightarrow 4y = 0 \Rightarrow y = 0$ (бһўбҹҗбһҖбҹ’бһҹбһўбһ¶бһ”бҹӢбһҹбҹҠбһёбһҹ)бҹ”
бҹўбҹ бҹӨ. бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{1+2i}{i^{15}}$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢ $a+bi$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $i^{15} = i^{12} \cdot i^3 = 1 \cdot (-i) = -i$бҹ”
  • бһҖбһ“бҹ’бһҹбҹ„бһҳбһ‘бҹ…бһҮбһ¶ $\frac{1+2i}{-i} = \frac{(1+2i) \cdot i}{-i \cdot i} = \frac{i + 2i^2}{1} = -2 + i$бҹ”
бҹўбҹ бҹҘ. бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸ (бһҳбһјбһҢбһ»бһӣ) бһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = (1-i\sqrt{3})(1+i)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһјбһҢбһ»бһӣбҹ– $|z| = |1-i\sqrt{3}| \cdot |1+i|$бҹ”
  • $|1-i\sqrt{3}| = \sqrt{1+3} = 2$ бһҜ $|1+i| = \sqrt{1+1} = \sqrt{2}$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $|z| = 2\sqrt{2}$бҹ”
бҹўбҹ бҹҰ. бһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -2 - 2i$ бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ…бһҳбҹ’бһ”бһ„бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $|z| = \sqrt{(-2)^2+(-2)^2} = \sqrt{8} = 2\sqrt{2}$бҹ” бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹЈ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ»бҹҶбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бһӮбһә $-\frac{3\pi}{4}$бҹ”
бҹўбҹ бҹ§. бһ”бһҫ $\arg(z) = \frac{\pi}{3}$ бһҸбһҫбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ“бҹғбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ $\arg(\overline{z})$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһӮбҹ’бһҡбһ№бҹҮбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ“бҹғбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһәбһ•бҹ’бһ‘бһ»бһҷбһ“бһ№бһ„бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһҠбһҫбһҳбҹ– $\arg(\overline{z}) = -\arg(z) = -\frac{\pi}{3}$бҹ”
бҹўбҹ бҹЁ. бһ”бһҳбҹ’бһӣбҹӮбһ„ $z = -4$ бһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z = -4$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһўбҹҗбһҖбҹ’бһҹбһ–бһ·бһҸбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһјбһҢбһ»бһӣ $r = |-4| = 4$ бһ“бһ·бһ„бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢ $\theta = \pi$бҹ”
бҹўбҹ бҹ©. бһҖбҹҶбһҺбһҸбҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһ“бҹғ $z = 1 + \cos \alpha + i\sin \alpha$ бһ…бҹҶбһ–бҹ„бҹҮ $0 < \alpha < \pi$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһ»бҹҶбһҢбһ»бһ”бҹ– $1 + \cos \alpha = 2\cos^2 \frac{\alpha}{2}$ бһ“бһ·бһ„ $\sin \alpha = 2\sin \frac{\alpha}{2}\cos \frac{\alpha}{2}$бҹ”
  • бһҠбһ¶бһҖбҹӢ $2\cos \frac{\alpha}{2}$ бһҮбһ¶бһҖбһҸбҹ’бһҸбһ¶бһҡбһҪбһҳ бһ”бһ¶бһ“бҹ– $2\cos \frac{\alpha}{2} \left(\cos \frac{\alpha}{2} + i\sin \frac{\alpha}{2}\right)$бҹ”
бҹўбҹЎбҹ . бһӮбһҺбһ“бһ¶ $\frac{z_1 \cdot z_2}{z_3}$ бһ”бһҫбһӮбҹ’бһҡбһ”бҹӢбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһҹбҹ’бһҳбһҫ $1$ бһ“бһ·бһ„бһҳбһ¶бһ“бһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһҡбҹҖбһ„бһӮбҹ’бһ“бһ¶ $\frac{\pi}{2}, \frac{\pi}{3}, \frac{\pi}{6}$бҹ” бһҡбһҖбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ„бҹӢбһ“бҹғбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\arg\left(\frac{z_1 \cdot z_2}{z_3}\right) = \arg(z_1) + \arg(z_2) - \arg(z_3) = \frac{\pi}{2} + \frac{\pi}{3} - \frac{\pi}{6}$бҹ”
  • бһҸбһҳбҹ’бһҡбһјбһңбһ—бһ¶бһӮбһ”бҹӮбһ„бһҡбһҪбһҳ бҹҰбҹ– $\frac{3\pi + 2\pi - \pi}{6} = \frac{4\pi}{6} = \frac{2\pi}{3}$бҹ”
бҹўбҹЎбҹЎ. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1 - i)^{10}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(1-i)^2 = -2i$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $((1-i)^2)^5 = (-2i)^5 = -32i^5$бҹ”
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $i^5 = i$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһә $-32i$бҹ”
бҹўбҹЎбҹў. бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $\left(\cos \frac{\pi}{10} + i\sin \frac{\pi}{10}\right)^{-5}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһҠбһәбһҳбҹҗбһҡбҹ– $\cos\left(-5 \cdot \frac{\pi}{10}\right) + i\sin\left(-5 \cdot \frac{\pi}{10}\right) = \cos\left(-\frac{\pi}{2}\right) + i\sin\left(-\frac{\pi}{2}\right) = 0 - i = -i$бҹ”
бҹўбҹЎбҹЈ. бһ”бһҫ $z = \cos \theta + i\sin \theta$ бһҸбһҫбһҖбһ“бҹ’бһҹбҹ„бһҳ $z^n + \frac{1}{z^n}$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһўбҹ’бһңбһё? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z^n = \cos n\theta + i\sin n\theta$ бһ“бһ·бһ„ $\frac{1}{z^n} = z^{-n} = \cos n\theta - i\sin n\theta$бҹ”
  • бһ”бһјбһҖбһҖбһ“бҹ’бһҹбҹ„бһҳбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ…бһјбһӣбһӮбҹ’бһ“бһ¶ бһ•бҹ’бһ“бҹӮбһҖбһҹбҹ’бһҡбһҳбһҫбһҹбҹ’бһҡбһҳбҹғбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһўбһҹбҹӢбҹ– $2\cos n\theta$бҹ”
бҹўбҹЎбҹӨ. бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\left(\frac{\sqrt{3}+i}{2}\right)^{2026}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҸбҹ’бһҡбһёбһҖбҹ„бһҺбһҳбһ¶бһҸбҹ’бһҡбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһӮбһә $\cos \frac{\pi}{6} + i\sin \frac{\pi}{6}$бҹ”
  • бһҸбһ¶бһҳбһҠбһәбһҳбҹҗбһҡбҹ– $\cos \frac{2026\pi}{6} + i\sin \frac{2026\pi}{6} = \cos \frac{1013\pi}{3} + i\sin \frac{1013\pi}{3}$бҹ”
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $1013 = 3 \cdot 337 + 2$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһңбһ¶бһҹбҹ’бһҳбһҫбһ“бһ№бһ„ $\cos\left(337\pi + \frac{2\pi}{3}\right)$ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҳбһ»бҹҶбһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһё бҹў бһӮбһә $\cos \frac{2\pi}{3} = -\frac{1}{2}$ бһ“бһ·бһ„ $\sin \frac{2\pi}{3} = \frac{\sqrt{3}}{2}$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮбһ…бһҳбҹ’бһӣбһҫбһҷбһӮбһә $-\frac{1}{2} + i\frac{\sqrt{3}}{2}$бҹ”
бҹўбҹЎбҹҘ. бһҸбһҫбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = (\cos \theta + i\sin \theta)^n$ бһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҳбһјбһҢбһ»бһӣбһ“бҹғ $\cos \theta + i\sin \theta$ бһӮбһә $\sqrt{\cos^2\theta + \sin^2\theta} = 1$бҹ”
  • бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһјбһҢбһ»бһӣбҹ– $|z| = |\cos \theta + i\sin \theta|^n = 1^n = 1$бҹ”
бҹўбҹЎбҹҰ. бһҡбһҖбһ«бһҹбһҖбһ¶бһҡбҹҒбһ“бҹғбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… $z = -8 - 6i$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһ„ $w = x+yi$ бһҮбһ¶бһ«бһҹбһҖбһ¶бһҡбҹҒ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбһёбһҖбһ¶бһҡбҹ– $x^2-y^2 = -8$, $2xy = -6$, бһ“бһ·бһ„ $x^2+y^2 = \sqrt{(-8)^2+(-6)^2} = 10$бҹ”
  • бһ”бһјбһҖбһҹбһҳбһёбһҖбһ¶бһҡбһ‘бһё бҹЎ бһ“бһ·бһ„бһ‘бһё бҹЈбҹ– $2x^2 = 2 \Rightarrow x = \pm 1$бҹ” бһҠбһҖбһҹбһҳбһёбһҖбһ¶бһҡбҹ– $2y^2 = 18 \Rightarrow y = \pm 3$бҹ”
  • бһҠбҹ„бһҷбһҹбһ¶бһҡ $2xy = -6 < 0$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $x$ бһ“бһ·бһ„ $y$ бһҳбһ¶бһ“бһҹбһүбҹ’бһүбһ¶бһ•бҹ’бһ‘бһ»бһҷбһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһ“бҹҒбҹҮбһ«бһҹбһҖбһ¶бһҡбҹҒбһӮбһә $\pm (1-3i)$бҹ”
бҹўбҹЎбҹ§. бһ”бһҫ $\omega$ бһҮбһ¶бһ«бһҹбһӮбһјбһ”бһҳбһ·бһ“бһ–бһ·бһҸбһ“бҹғ $1$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1+\omega)(1+\omega^2)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҷбһҫбһ„бһҳбһ¶бһ“ $1 + \omega + \omega^2 = 0 \Rightarrow 1+\omega = -\omega^2$ бһ“бһ·бһ„ $1+\omega^2 = -\omega$бҹ”
  • бһӮбһ»бһҺбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бҹ– $(-\omega^2)(-\omega) = \omega^3 = 1$бҹ”
бҹўбҹЎбҹЁ. бһҸбһҫбһ…бҹҶбһҺбһ»бһ…бһҸбҹҶбһҺбһ¶бһ„бһ«бһҹбһ‘бһё $n$ бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғ $1$ бһҹбҹ’бһҗбһ·бһҸбһ“бҹ…бһӣбһҫбһҒбҹ’бһҹбҹӮбһҖбҹ„бһ„бһ’бһҡбһҺбһёбһҳбһ¶бһҸбҹ’бһҡбһҳбһҪбһҷбһҺбһ¶? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһӮбҹ’бһҡбһ”бҹӢбһ«бһҹбһ‘бһё $n$ бһ“бҹғ $1$ бһҹбһ»бһ‘бҹ’бһ’бһҸбҹӮбһҳбһ¶бһ“бһҳбһјбһҢбһ»бһӣбһҹбҹ’бһҳбһҫ $1$ бһҮбһ¶бһ“бһ·бһ…бҹ’бһ… бһҠбһјбһ…бһ“бҹҒбҹҮбһ…бҹҶбһҺбһ»бһ…бһҸбҹҶбһҺбһ¶бһ„бһҡбһ”бһҹбҹӢбһңбһ¶бһҳбһ¶бһ“бһ…бһҳбҹ’бһ„бһ¶бһҷбһ–бһёбһӮбһӣбҹӢбһҸбһҳбҹ’бһҡбһ»бһҷбһҹбҹ’бһҳбһҫ бҹЎ бһҠбҹӮбһӣбһҮбһ¶бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ“бҹғбһҡбһ„бҹ’бһңбһ„бҹӢбһҜбһҖбһҸбһ¶бҹ”
бҹўбҹЎбҹ©. бһҡбһҖбһ•бһӣбһӮбһ»бһҺбһ“бҹғбһ«бһҹбһӮбһјбһ”бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹғ $1$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ«бһҹбһӮбһјбһ”бһ‘бһ¶бҹҶбһ„бһ”бһёбһӮбһә $1, \omega, \omega^2$ бһ“бһ¶бҹҶбһұбҹ’бһҷбһ•бһӣбһӮбһ»бһҺбһӮбһә $1 \cdot \omega \cdot \omega^2 = \omega^3 = 1$бҹ”
бҹўбҹўбҹ . бһ”бһҫ $z_k = \cos \frac{2k\pi}{3} + i\sin \frac{2k\pi}{3}$ бһ…бҹҶбһ–бҹ„бҹҮ $k=0,1,2$ бһҮбһ¶бһ«бһҹбһӮбһјбһ”бһ“бҹғ $1$ бһҸбһҫбһ«бһҹбһҺбһ¶бһҒбҹ’бһӣбҹҮбһҮбһ¶бһ«бһҹбһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһӮбҹ’бһ“бһ¶? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $z_0 = 1$ (бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ)бҹ” $z_1 = \cos \frac{2\pi}{3} + i\sin \frac{2\pi}{3} = -\frac{1}{2} + i\frac{\sqrt{3}}{2}$бҹ”
  • $z_2 = \cos \frac{4\pi}{3} + i\sin \frac{4\pi}{3} = -\frac{1}{2} - i\frac{\sqrt{3}}{2}$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $z_1$ бһ“бһ·бһ„ $z_2$ бһҮбһ¶бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһӮбҹ’бһ“бһ¶бҹ”
бҹўбҹўбҹЎ. бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - 2z + 2 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $\Delta' = (-1)^2 - 1(2) = 1 - 2 = -1 = i^2$бҹ” бһҠбһјбһ…бһ“бҹҒбҹҮ $z = 1 \pm i$бҹ”
бҹўбҹўбҹў. бһ”бһҫ $z^2 - 6z + c = 0$ бһҳбһ¶бһ“бһ«бһҹбһҳбһҪбһҷбһӮбһә $3 + 5i$ бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҳбҹҒбһӮбһ»бһҺбһ–бһ·бһҸ $c$ бһӮбһәбҹ– (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҠбҹ„бһҷбһҹбһ¶бһҡбһҳбҹҒбһӮбһ»бһҺбһҮбһ¶бһ…бҹҶбһ“бһҪбһ“бһ–бһ·бһҸ бһ«бһҹбһҳбһҪбһҷбһ‘бҹҖбһҸбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҮбһ¶бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбһә $3 - 5i$бҹ”
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸ бһ•бһӣбһӮбһ»бһҺбһ«бһҹ $P = \frac{c}{a} = c \Rightarrow c = (3+5i)(3-5i) = 3^2 + 5^2 = 9 + 25 = 34$бҹ”
бҹўбҹўбҹЈ. бһҡбһҖбһҹбҹҶбһҺбһ»бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^4 - 16 = 0$ бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһ»бҹҶбһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ…бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • $(z^2-4)(z^2+4) = 0 \Rightarrow z^2 = 4$ бһ¬ $z^2 = -4$бҹ”
  • бһ“бһ¶бҹҶбһұбҹ’бһҷбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷ $z = \pm 2$ бһ“бһ·бһ„ $z = \pm 2i$бҹ”
бҹўбҹўбҹӨ. бһҡбһҖбһ•бһӣбһ”бһјбһҖбһ“бҹғбһ«бһҹбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ“бҹғбһҹбһҳбһёбһҖбһ¶бһҡ $z^2 - (5-2i)z + 10 = 0$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһ”бһ‘бһңбҹҖбһҸ бһ•бһӣбһ”бһјбһҖбһ«бһҹ $S = -\frac{b}{a} = -\frac{-(5-2i)}{1} = 5 - 2i$бҹ”
бҹўбҹўбҹҘ. бһ”бһҫ $z^2 - z + 1 = 0$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $z^6$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбҹ–
  • бһ–бһёбһҹбһҳбһёбһҖбһ¶бһҡбһҠбһҫбһҳбһҷбһҫбһ„бһ”бһ¶бһ“ $z^3 = -1$ (бһҠбҹ„бһҷбһӮбһ»бһҺбһ“бһ№бһ„ $z+1$)бҹ”
  • бһҠбһјбһ…бһ“бҹҒбҹҮ $z^6 = (z^3)^2 = (-1)^2 = 1$бҹ”
